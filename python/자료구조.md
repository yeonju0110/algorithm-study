## 📑 스택

- 선입후출, 후입선출
- 재귀함수 -> DFS

```py
stack.append(5)
stack.pop()

print(stack) # 최하단 원소부터
print(stack[::-1]) # 최상단 원소부터
```

## 📑 큐

- 선입선출

```py
from collections import deque

queue = deque()

queue.append(5)
queue.popleft()

print(queue) # 먼저 들어온 순서대로
queue.reverse()
print(queue) # 나중에 들어온 순서대로
```
